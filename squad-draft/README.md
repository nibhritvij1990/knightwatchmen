Squad Draft - Prototype
=======================

This is a minimal Vite + React prototype for the Squad Draft app.

Quick start:

1. Install deps
   npm install

2. Run dev server
   npm run dev

Files included:
- src/App.jsx     (main app)
- src/main.jsx
- src/index.css
- index.html
- tailwind.config.cjs
- postcss.config.cjs
- package.json

Notes:
- The app persists to localStorage (key: squad_draft_v1).
- Export PNG/PDF uses html-to-image and jsPDF.
